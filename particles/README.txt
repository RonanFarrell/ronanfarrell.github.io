How to run:
    + Open particle.html in a web browser.
    + Click to set the initial position of the particle emitter, hold and drag
      to change the size of the particles emitted.
    + The following click determines the direction, the further away the click is,
      the faster the particles will travel.
    + Hold down the second click to modify the interval between particles,
      the bigger the circle, the bigger the interval between particles.


Supported browsers: Latest version of Chrome.
                    Tested in Chrome for Mac.

Additional features:
    + Multiple particle emitters can be created
    + Advanced canvas rendering techniques
