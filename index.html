ImploExplo2013

ImploExplo is a coop platformer developed for browsers using HTML5 & JavaScript. The game revolves around two characters, Implo and Explo, and their unique abilities. Implo can implode pulling objects, including Explo, towards it. Explo does the opposite, it can explode, pushing objects away from it.

PortaLESS2013

PortaLESS is a 3D puzzle games made with Ogre3D and Havok Physics. The game involves using a gravity gun to interact with the environment to solve various physics puzzles.

Particles2013

Particles was developed as part of a technical interview. It turned out pretty well so I decided to put it up here. It is written in JavaScript and uses the HTML5 canvas for rendering. It is a bit rough around the edges, it was made in about three hours, most of that involved developing the UI.

Turbo Trolley Racers2012

Turbo Trolley Racers was developed with Keith Cully during third year of college. Written in C++ and uses Ogre3D and PhysX. We displayed the game at the Festival International des Jeux, Cannes in February 2012.
